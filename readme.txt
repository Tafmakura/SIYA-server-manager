== Description ==
The SIYA Server Manager is a plugin for WooCommerce to integrate with server provisioning services, handle subscription status, and manage server connections.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/siya-server-manager` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
